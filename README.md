**HookMind** is a decentralized protocol that uses autonomous AI agents to manage and protect liquidity provision on Uniswap v4. By separating complex computation (off-chain AI) from execution (on-chain Hooks), HookMind solves the "Static Fee Problem" and provides the first automated **Impermanent Loss Insurance** for liquidity providers.

---

## 🚀 The Value Proposition

Liquidity providers on decentralized exchanges face two massive problems:
1. **Static Fees:** Fixed fee tiers cannot adapt to real-time market volatility, causing LPs to leave money on the table or lose value to arbitrageurs.
2. **Impermanent Loss:** LPs frequently lose value compared to simply holding the underlying assets, with no safety net or protection.

**HookMind solves both by combining AI and on-chain insurance.**

### Three Layers of Protection
1. **AI-Powered Dynamic Fees**: Our AI agents analyze market conditions every 12 seconds. When volatility spikes, the AI raises the trading fee to protect LPs. When the market stabilizes, it lowers fees to capture volume.
2. **Impermanent Loss Insurance**: LPs pay a 10 USDC premium when they add liquidity. If they exit with an impermanent loss greater than 2%, the protocol's insurance fund automatically compensates them.
3. **Transparent Audit Trail**: Every AI fee decision is permanently recorded on IPFS before it executes on-chain. Investors can cryptographically verify exactly *why* any fee was changed.

## 💰 Sustainable Revenue Model

HookMind is built as a highly scalable **B2B2C Software-as-a-Service (SaaS)** infrastructure, generating revenue through three sustainable streams:

1.  **Agent Activation Fees**: Users pay a one-time activation fee of **0.00244 ETH (~$5)** to register an AI agent. This creates a low-barrier entry for mass adoption while funding the protocol treasury.
2.  **Protocol Software Toll**: A **1% fee** is taken from the swap fees generated by HookMind-managed pools, creating a sustainable revenue stream that scales with Total Value Locked (TVL).
3.  **Insurance Premiums**: LPs opt-in to IL Insurance by paying a **10 USDC premium**, creating a self-sustaining pool for risk management.

**Unit Economics at Scale:** 10,000 active LPs paying 10 USDC/month = $100K/month in premiums alone, plus the 1% TVL-based software toll.

## ⚖️ Regulatory Compliance

HookMind is designed with a **Non-Custodial, Software-Only** approach to maximize global compliance and appeal to institutional stakeholders:

-   **Mexico (Ley Fintech)**: Operating under a strictly non-custodial model, HookMind provides software for decentralized execution without holding user funds, clearly aligning with the safe harbor for software providers rather than regulated ITFs.
-   **European Union (MiCA)**: Built for the July 2026 deadline. The protocol operates via "full decentralization" to fall under the Recital 22 exemption, utilizing IPFS for immutable audit trails and community-driven governance.
-   **Auditability**: Every AI decision is cryptographically signed via ECDSA and pinned to **IPFS**, ensuring a fully transparent and verifiable audit trail for compliance teams.

## Architecture

```text
Off-chain AI Agent (Node.js)
  ↓ reads Unichain block data every 12s
  ↓ computes volatility score (0–10,000) via PredictiveInventoryModel
  ↓ signs recommendation with ECDSA key
  ↓ pins decision to IPFS (audit trail)
  ↓ calls updateNeuralState() on HookMindCore

HookMindCore.sol (Uniswap v4 Hook)
  ├── beforeSwap()     → applies dynamic fee override via LPFeeLibrary
  ├── afterSwap()      → emits fee events, IL exposure check
  ├── afterAddLiquidity()   → enrollLp() on ILInsurance
  └── afterRemoveLiquidity() → processExit() on ILInsurance

ILInsurance.sol
  ├── payPremium()     → LP pays 10 USDC, position marked as insured
  ├── processExit()    → calculates IL exposure on removal
  └── claimInsurance() → epoch-prorated USDC payout
```

**Key technical properties:**
- `TSTORE`/`TLOAD` (EIP-1153) — passes dynamic fee between `beforeSwap` and `afterSwap` with zero SLOADs
- `BeforeSwapReturnDelta` — asymmetric spread when volatility > 7000/10000
- ECDSA + nonce replay protection — only registered agents can update pool state
- Flash loan guard — agent rejects data with >15% spot-to-TWAP deviation

## Quickstart

```bash
# 1. Clone and install
git clone https://github.com/hookmind/hookmind
pnpm install

# 2. Configure environment
cp .env.example .env.local
# Add: UNISWAP_API_KEY, AGENT_EOA_ADDRESS, etc.

# 3. Start the frontend
pnpm --filter web dev

# 4. Run the AI agent daemon (separate terminal)
pnpm run dev:agent
```
