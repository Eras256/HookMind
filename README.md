**HookMind** is a decentralized protocol of autonomous AI agents (Digital Managers) that optimize liquidity provision on Uniswap v4. By separating computation (off-chain AI) from execution (on-chain Hooks), HookMind solves the "Static Fee Problem" and provides the first automated **Impermanent Loss Insurance** for LPs.

---

## 🚀 Business Model: Scaling the Intelligence Mesh

HookMind is built as a **B2B2C Software-as-a-Service (SaaS)** infrastructure for the Uniswap ecosystem:

1.  **Registry Fees**: Users pay a one-time activation fee of **0.00244 ETH** to register their agent node. This creates a low-barrier entry for mass adoption while funding the protocol treasury.
2.  **Performance Tolls**: A small percentage (e.g., 5 bps) of the "excess yield" generated by the AI's dynamic fee adjustments is routed to the protocol.
3.  **Insurance Premiums**: LPs can opt-in to the **IL Insurance** by paying a premium in USDC, creating a self-sustaining pool for risk management.

## ⚖️ Regulatory Compliance

HookMind is designed with a **Non-Custodial, Software-Only** approach to maximize global compliance:

-   **Mexico (Ley Fintech)**: Operating under a non-custodial model, HookMind provides software for decentralized execution without holding user funds, aligning with the "gray zone" of software providers rather than centralized ITFs.
-   **European Union (MiCA)**: Prepared for the July 2026 deadline. The protocol aims for "full decentralization" to fall under the Recital 22 exemption, utilizing IPFS for audit trails and community-driven governance.
-   **Auditability**: Every AI decision is signed via ECDSA and pinned to **IPFS**, ensuring a transparent "Cognitive Audit Trail" for institutional stakeholders.

## Architecture

```
Off-chain AI Agent (Node.js)
  ↓ reads Unichain block data every 12s
  ↓ computes volatility score (0–10,000) via PredictiveInventoryModel
  ↓ signs recommendation with ECDSA key
  ↓ pins decision to IPFS (audit trail)
  ↓ calls updateNeuralState() on HookMindCore

HookMindCore.sol (Uniswap v4 Hook)
  ├── beforeSwap()     → applies dynamic fee override via LPFeeLibrary
  ├── afterSwap()      → emits fee events, IL exposure check
  ├── afterAddLiquidity()   → enrollLp() on ILInsurance
  └── afterRemoveLiquidity() → processExit() on ILInsurance

ILInsurance.sol
  ├── payPremium()     → LP pays 10 USDC, position marked as insured
  ├── processExit()    → calculates IL exposure on removal
  └── claimInsurance() → epoch-prorated USDC payout
```

**Key technical properties:**
- `TSTORE`/`TLOAD` (EIP-1153) — passes dynamic fee between `beforeSwap` and `afterSwap` with zero SLOADs
- `BeforeSwapReturnDelta` — asymmetric spread when volatility > 7000/10000
- ECDSA + nonce replay protection — only registered agents can update pool state
- Flash loan guard — agent rejects data with >15% spot-to-TWAP deviation

## Legal & Compliance

HookMind operates as a **non-custodial DeFi protocol**. Smart contracts are immutable and user-controlled. The protocol does not hold or manage user funds directly.

- **Mexico (Ley Fintech 2018):** Non-custodial DeFi protocols that do not offer fiat on/off ramps fall outside mandatory CNBV registration requirements. HookMind does not custody assets.
- **EU (MiCA 2024):** Non-custodial DeFi protocols are explicitly excluded from MiCA's licensing requirements under Art. 2(4).
- **US:** Protocol operates as non-custodial software. No securities offering is made.
